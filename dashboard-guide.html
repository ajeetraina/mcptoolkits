Understanding the MCPToolkits Dashboard

This guide explains the metrics and information displayed on your MCPToolkits dashboard.

Note: The current dashboard shows sample data to demonstrate functionality. In your production environment, it will display real data from your MCP servers.

Overview Section

The top section of the dashboard displays key metrics about your MCP server infrastructure:

Active Servers

4

Active Servers: Shows the number of MCP servers that are currently online and responding. This helps you quickly see the overall health of your infrastructure.

Warnings

3

Warnings: The total number of issues requiring attention across your MCP infrastructure, including both configuration issues and performance warnings.

Avg. Latency

23ms

Average Latency: The average response time across all your active MCP servers, measured in milliseconds. Lower values indicate better performance.

Active Users

571

Active Users: The total number of users currently connected to and using your MCP servers. Helps track usage patterns and capacity planning.

Server Status Table

The server status table provides detailed information about each individual MCP server in your infrastructure:

  • Server: The name of the MCP server
  • Status: Current server state
    • Online - Server is operating normally
    • Warning - Server is experiencing issues but is still operational
    • Offline - Server is not responding or unavailable
  • Latency: Response time measured in milliseconds - lower is better
  • Uptime: Percentage of time the server has been available
  • Region: Geographic location of the server

Configuration Status

The configuration status chart shows how many of your MCP server configurations are properly synchronized:

  • Synced: Number and percentage of configurations that are in sync across your MCP servers
  • Out of Sync: Number and percentage of configurations that need to be synchronized

The "Sync All Configurations" button allows you to synchronize all configurations with a single click, ensuring consistency across your entire MCP infrastructure.

Performance Charts

The dashboard includes charts displaying performance metrics over time:

  • Performance Chart: Shows latency trends over the past 24 hours, helping you identify performance patterns and potential issues
  • User Activity Chart: Shows the number of active users over time, helping you understand usage patterns and capacity requirements

Recent Activity

The Recent Activity section displays the latest events and alerts from your MCP servers, such as:

  • Configuration changes and syncs
  • Performance warnings
  • Server status changes
  • System updates and maintenance

Navigation Tabs

The dashboard has four main navigation tabs at the top:

  • Overview: The main dashboard view with key metrics and status information
  • Servers: Detailed view of all your MCP servers with in-depth metrics and management options
  • Configuration: Advanced configuration management for your MCP servers
  • Analytics: Detailed performance analytics and reports

Note: This is a demo version of the dashboard. In your production environment, all tabs will be fully functional with your actual MCP server data.
